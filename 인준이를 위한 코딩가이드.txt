알아두면 쓸데있는 코딩 지식 (알쓸코지)

###
# 꼭 알아두도록 하자!!!
###

#1
- node_modules / package-lock.json 파일은 지워도 된다.
  npm i 로 다시 깔 수 있다.

#2
- 코딩을 할 땐 기본적으로 들여쓰기를 Tab으로한다.
  들여쓰기를 맞추는 습관을 들이자!
  자동 들여쓰기는 전체 선택을 하고 Ctrl + K, Ctrl + F 를 차례대로 눌러주면 된다.

#3
- JSON 파일은 키-값 형태로 되어 있어서
  object.key로 키를 참조하면 값을 받을 수 있다.

#4
- JSON의 형태에는 배열, 객체가 있는데,
  대괄호([])로 둘러 쌓인 부분은 배열이라 index로 참조.
  중괄호({})로 둘러 쌓인 부분은 객체라 key로 참조.

#5
- shift + tab 키로 들여쓰기를 취소할 수 있다.

#6
- 웹 개발 시에는 항상 개발자 도구를 활용하자.

#7
- 개발자 도구에서 네트워크 탭을 통해 API가 잘 갔는지 알 수 있다.

#8
- 개발하면서 값이 이상하게 들어간다 싶으면 console.log를 찍어보자.
